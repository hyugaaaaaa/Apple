ヘルプ

iPhone を Mac の
左手デバイスに。

Left Controller は、iPhone をワイヤレスのアプリ起動ランチャーとして Mac の隣に置いて使う PWA アプリです。タップひとつで Mac のアプリが起動・最小化されます。

01はじめての方へ

1. Mac 側のセットアップ

Mac の Safari で /mac-setup を開き、エージェント用のセットアップスクリプトをダウンロードしてターミナルで実行します。

  • ホームフォルダに ~/.left-controller/ が作られます
  • Mac エージェントが launchd で常駐し、ログイン時に自動起動します
  • セットアップ完了画面に「接続PIN」が表示されます。必ず控えてください

2. iPhone 側のセットアップ

iPhone の Safari で同じ URL を開き、画面下部の 共有 → ホーム画面に追加 で PWA を追加します。

  • ホーム画面のアイコンから起動すると認証画面が表示されます
  • Mac 側で表示された接続PIN を入力します
  • 認証後、コントローラー画面に遷移します

3. アプリの登録（Mac 側）

Mac の Safari で /admin.html を開き、PIN でログインします。

  • 「アプリ」タブから登録したいアプリを選んでスロットに割り当て
  • 最大 24 個（8アプリ × 3ページ）まで登録可能
  • 「保存」を押すと iPhone 側にすぐ反映されます

02コントローラーの使い方

👆
1回タップ アプリを起動し、フォアグラウンドに表示。他のウィンドウは自動的に最小化されます。
👆👆
ダブルタップ（320ms 以内に2回） そのアプリのウィンドウを Dock へ最小化。再度タップするとフォアグラウンドに戻ります。
←→
左右スワイプ / ページ送り ページを切り替えて、登録した別のアプリ群を表示します。

03管理画面の操作

接続PIN の確認・再発行

管理画面の ⓘ 情報 タブから現在の PIN を確認できます。「PIN再発行」で新しいPIN に切り替え可能です。

PIN を忘れた場合は、ログイン画面の「PINを忘れた場合」→「PINを確認する」で Mac エージェント経由で再表示できます（エージェントが起動中である必要があります）。

登録アプリの並べ替え

「アプリ」タブのスロットをタップすると、別のアプリと入れ替え可能です。空きスロットには新規アプリを追加できます。

04困ったときは

📡iPhone から接続できない

Mac エージェントが起動していない可能性があります。Mac のターミナルで以下を実行して状態を確認してください：

launchctl print gui/$(id -u)/com.leftcontroller.mac-agent | grep state

state = running でない場合は、以下で再起動できます：

launchctl kickstart -k gui/$(id -u)/com.leftcontroller.mac-agent

🪟ダブルタップで最小化できないアプリがある

Mac の Accessibility 権限が必要なアプリ（Electron 系を含む一部のアプリ）があります。通常は付属の Swift ヘルパー（~/.left-controller/leftctl-minimize）が AX API 経由で動作しますが、まれに失敗する場合は次を確認してください：

  • Mac エージェントが最新版か（launchctl print で確認）
  • leftctl-minimize が存在するか（ls ~/.left-controller/）
🛡️許可ダイアログが大量に出る

初回のみ macOS が各アプリの操作許可を求めます。Swift ヘルパーは AX API を使用するため、通常は明示的な許可なしに動作しますが、AppleScript フォールバックを使う場合は「自動化」権限のダイアログが表示されることがあります。

📲登録したアプリが iPhone 側に表示されない

管理画面で「保存」ボタンを押したか確認してください。保存後は数秒以内に iPhone 側のコントローラーに反映されます。

反映されない場合は、iPhone 側で PWA をいったん閉じて再起動するか、コントローラー画面下部のデバッグから「キャッシュクリア」を実行してください。

🔐認証後にコントローラー画面に遷移しない

PIN を正しく入力したかを確認してください。5回間違えると 10 分間ロックされます。ロック解除を待つか、管理画面から PIN を再発行してください。

🖥️複数の Mac で使いたい

各 Mac ごとに別の DEVICE_ID を設定し、それぞれセットアップを実行してください。リレー側で MAC_PINS 環境変数を "mac1:111111,mac2:222222" のように設定すれば、Mac ごとに固定 PIN を割り当てられます。
